# RFID_Reader - Basic ESP8266 RFID reader that display results on a Nokia 5110 Display 


## Pinout used 

</br>


### RF522

| RFID-RF522 | ESP8266  |
| :---       |:---      |
| SDA (SS)   | D4       |
| SCK        | D5       |
| MOSI       | D7       |
| MISO       | D6       |
| IRQ        | -        |
| GND        | G        |
| RST        | D3       |
| 3.3V (VCC) | 3V3      |
 
</br>
</br>
</br>

### LCD Nokia 5110

| LCD 5110   | ESP8266  |
| :---       |:---      |
| RST        | D0       | - D0
| CE         | D1       | - D1
| DC         | D2       | - D2
| DIN        | D5       | - D3
| CLK        | D7       | - D4
| VCC        | 3V3      |
| BL         | 3V3      |
| GND        | G        |
