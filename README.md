# RFID_Reader - Basic ESP8266 RFID reader that display results on a Nokia 5110 Display 


### Pinout used 

| RFID-RF522 | ESP8266  |
| :---       |:---      |
| SDA (SS)   | D4       |
| SCK        | D5       |
| MOSI       | D7       |
| MISO       | D6       |
| IRQ        | -        |
| GND        | G        |
| RST        | D3       |
| 3.3V (VCC) | 3V       |
